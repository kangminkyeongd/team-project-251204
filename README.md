# team-project-251204
❄눈송이 다이어리 웹사이트

1. 프로젝트 개요
하루의 감정을 시각적으로 기록하는 웹 기반 다이어리 서비스인 「눈송이 다이어리」를 제작하는 것을 목표로 하였다. 일기를 글로만 남기는 기존 방식에서 벗어나, 하루를 눈송이 하나로 표현하여 시간이 쌓이는 과정을 직관적으로 보여주는 것이 본 프로젝트의 핵심 개념이다. 특히 겨울과 크리스마스 분위기를 살려, 감정 기록과 계절적 감성을 함께 담아내고자 하였다.

2. 기획 의도
- 기록에 대한 부담을 줄이기하루를 길게 쓰지 않아도, 감정 선택과 간단한 동작만으로 기록이 남도록 설계하였다.
- 감정을 시각적으로 체험감정에 따라 눈송이의 속도, 크기, 배경 분위기가 달라져
사용자가 자신의 기분을 화면으로 느낄 수 있도록 하였다.
- 겨울과 크리스마스 감성 강화눈 내리는 화면과 함께 크리스마스 느낌의 음악을 더해
감정 기록을 하나의 계절적 경험으로 만들고자 하였다.

3. 주요 기능 설명

3.1 감정 선택 기반 눈송이 생성
사용자는 일기를 작성할 때 오늘의 감정(총 10종)을 선택할 수 있다.선택된 감정에 따라 요소들(눈송이의 낙하 속도, 눈송이의 크기, 화면 배경 색감, 전체 분위기 및 시각적 효과)이 달라진다.
이를 통해 각 하루의 감정이 서로 다른 형태의 눈송이로 표현된다.

3.2 눈송이 누적 및 기록 표현
일기 한 편 = 눈송이 1개
일기를 작성할 때마다 눈송이가 화면 위에서 아래로 떨어진다.
떨어진 눈송이는 유리병에 저장된 기록의 개수로 환산된다.
유리병은 사용자가 겨울 동안 쌓아온 기록을 상징하며,눈송이 개수는 곧 사용자가 기록을 이어온 시간의 결과를 의미한다.

3.3 크리스마스 음악 ON/OFF 기능
화면에 표시된 소리(음악) 아이콘을 클릭하면
크리스마스 분위기의 배경 음악이 재생되고, 다시 클릭 시 음악이 정지된다.
사용자는 상황에 따라 소리를 켜거나 끌 수 있어조용한 환경에서도 불편 없이 사용할 수 있다.
이 기능을 통해 사용자는 시각뿐만 아니라 청각적으로도 겨울 감성을 체험할 수 있다.

4. 기대 효과 및 의의
눈송이 다이어리는 단순한 기록 도구를 넘어, 감정과 계절, 시간을 함께 저장하는 인터랙티브 다이어리이다. 기록의 진입 장벽을 낮추고, 감정을 시각·청각적으로 표현하며 하루하루를 눈송이로 쌓아가는 경험을 제공한다. 이를 통해 사용자는 자연스럽게 기록을 지속하게 되며, 자신의 감정 변화를 돌아볼 수 있는 계기를 얻을 수 있다.
